⚡ البدء السريع (5 دقائق فقط):

  1. تحقق من تثبيت: Java 17+, Maven 3.6+, MySQL 8.0+
  
  2. أنشئ قاعدة البيانات:
     mysql -u root -p
     CREATE DATABASE devboard;
     CREATE USER 'devboard_user'@'localhost' IDENTIFIED BY 'DevBoard@123Secure!';
     GRANT ALL PRIVILEGES ON devboard.* TO 'devboard_user'@'localhost';
     FLUSH PRIVILEGES;
  
  3. انسخ الملفات إلى مشروعك
     (اتبع الإرشادات في INSTALLATION_GUIDE.md)
  
  4. شغّل:
     mvn clean install
     mvn spring-boot:run
  
  5. الوصول:
     Backend:  http://localhost:8080
     Swagger:  http://localhost:8080/api/swagger-ui.html
